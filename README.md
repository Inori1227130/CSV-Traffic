# 交通事故データ分析アプリ

2023年の全国交通事故データ(CSV)を分析し、  
Flask を使って条件別に表示できるようにした Web アプリです。

## 使用技術
- Python (Flask)
- pandas
- HTML/CSS

## 注意
今回は警察庁のオープンデータを使用していますが、容量圧迫のため、GitHubにCSVファイルは入っておりません。こちらのURLから本票の方をダウンロードし、dataフォルダに貼り付けてください。
https://www.npa.go.jp/publications/statistics/koutsuu/opendata/2023/opendata_2023.html
